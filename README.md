**GreenWave Conference App**

The GreenWave Conference App is a simple system designed to help attendees and administrators manage conference activities efficiently.

**Features**

**For Attendees**
  - Register for an account and log in.
  - Book conference passes.
  - Register for available exhibitions.
  - Sign up for workshops within each exhibition.

**For Administrators**
  - Manage all exhibitions and workshops.
  - View attendee registrations.
  - Add, update, or remove workshops and exhibitions.
